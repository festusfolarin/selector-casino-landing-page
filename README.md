# 🎯 Landing Page Assessment – Selector Casino Inspired

## 📌 Project Overview

This project is a **conversion-focused landing page** built as part of a technical assessment.

The goal was to recreate the **look and feel of a real product website** (Selector Casino) while maintaining a simplified structure and prioritizing:

- Clean, maintainable code
- High performance
- Strong conversion design

---

## 🚀 Objectives

- Replicate the **visual style** of the reference website (colors, layout, UI elements)
- Build a **mobile-first, responsive** landing page
- Focus on **user engagement and conversions** (CTA buttons, promo offers, content clarity)
- Deliver production-quality frontend code without reliance on frameworks

---

## 📊 Success Criteria

This project was developed with the following KPIs in mind:

- ✅ Delivered within the required timeline
- ✅ Clean, readable, and maintainable code structure
- ✅ Optimized for performance (Lighthouse & Core Web Vitals)
- ✅ Built with minimal reliance on external tools
- ✅ Demonstrates ability to work independently

---

## 🧱 Page Structure

The landing page follows a clear, conversion-driven structure:

1. **Header**
   - Logo
   - Navigation / CTA buttons

2. **Hero / Offer Section**
   - Main promotional message
   - Bonus offer details
   - Promo code
   - Primary call-to-action

3. **Content Section**
   - Informational text
   - Comparison / bonus table
   - Supporting visuals (screenshots)

4. **Footer**
   - Additional links / disclaimers

---

## 🛠️ Tech Stack

- **HTML5** — Semantic structure (`header`, `main`, `section`, `footer`)
- **CSS3** — Custom styling (no frameworks)
- **JavaScript (Vanilla)** — Interactive behavior

---

## ⚙️ Development Approach

### 📱 Mobile-First

- Designed and built starting from small screens upward
- Responsive across mobile, tablet, and desktop

### 🎨 Layout Techniques

- **Flexbox** for standard layouts
- **CSS Grid** for more complex sections (where necessary)

### 🚫 Constraints Followed

- No CSS frameworks (Bootstrap, Tailwind, etc.)
- No jQuery or external JS libraries
- No page builders or templates
- No outdated layout methods (floats, table-based design)

---

## ⚡ Performance Considerations

- Optimized layout and styles for fast rendering
- Minimal DOM complexity
- Efficient CSS structure
- Lightweight JavaScript
